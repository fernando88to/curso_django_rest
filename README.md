### Projeto de estudo do python api

### Requisitos

* pip
* python 3

### Processo de configuração inicial

```
python3 -m venv ./venv
source venv/bin/activate
pip install django
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
```


### Rodar o servidor

```
python manage.py runserver
```
